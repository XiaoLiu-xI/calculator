# 计算器

一个简洁、易用的 Android 计算器应用，支持基础算术运算，界面清爽，响应快速。

## 项目介绍

本项目是基于 Android + Java 开发的轻量级计算器应用，实现日常所需的加减乘除等基础计算功能，适合学习 Android 基础 UI 与逻辑交互。

## 功能特性

- 支持加、减、乘、除基础运算
- 支持小数点输入与清空操作
- 简洁直观的按钮布局
- 实时显示输入与计算结果
- 无广告、轻量化、启动快

## 技术栈

- 语言：Java
- 平台：Android
- 构建工具：Gradle

## 项目结构

```
app/                  # 主模块源码
.gitignore            # Git 忽略文件
build.gradle          # 项目构建配置
gradle.properties     # Gradle 配置
gradlew / gradlew.bat # Gradle 包装器
local.properties      # 本地配置
settings.gradle       # 项目设置
README.md             # 项目说明
```

## 安装与运行

1. 克隆项目到本地

```bash
git clone https://github.com/XiaoLiu-xI/calculator.git
```

2. 使用 Android Studio 打开项目
3. 连接设备或启动模拟器
4. 点击 Run 运行应用
